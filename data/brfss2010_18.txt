# Data
This project uses data derived from the Behavioral Risk Factor Survelliance System (BRFSS) which was provided with help by Professor Partha Deb. 
Years used for the study: 2010-2018
To replicate analysis: 

1. Download annual BRFSS datasets from
   https://www.cdc.gov/brfss/annual_data/annual_data.htm
2. Clean each year using individual do file to their corresponding year.
4. Combine data sets in chunks using AppendedData do file to create:
   brfss_2010_2018_combined.dta
5. Place file in data.
6. Run files in analysis folder.
